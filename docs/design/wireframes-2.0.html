Well Done 2.0 — “Clean Craft” wireframes

Reference wireframes for the Phase 1 screen rebuilds (see docs/requirements/14-design-direction.md). Revised 2026-07-18: the warm-paper canvas is retired — neutral white base, hairline card borders, subtler shadows; terracotta/honey/sage survive only as meaning-bearing accents (actions, streaks, success). Layout + tone are the point; spacing and copy are directional. New-in-2.0 elements shown: light large-title headers (dark navy retired), frosted full-width tab bar (v1 shape, modern treatment), year switcher on the dashboard, class-point shortcut, editable student header, attribution chips, bento stats, and the neutral dark mode. Emoji rule: emoji appears only as student avatar data. The tab bar uses real lucide line icons (as in v1); remaining flame/camera/trash glyphs are placeholders for their lucide equivalents; all other UI is text-only.
